# Go Github Activity
Learning demo project created in Go to fetch user activity from Github via cli.

Sample solution for the [github-user-activity](https://roadmap.sh/projects/github-user-activity) challenge from [roadmap.sh](https://roadmap.sh/dashboard).

## Usage
Install the app using `go install github.com/saga-sanga/go-github-activity`

In the CLI run `go-github-activity <username>`
